Adicionar transação

O programa é básico e bem simples de usar, sendo necessário apenas adicionar transações de receitas e despesas com seus respectivos nomes e valores, utilizando um sinal de menos para representar as despesas, por exemplo: "Nome da transação: cartão de crédito, valor da transação: -400,00". Além disso, o programa calcula automaticamente o saldo atual com base na soma total de receitas e despesas. Para armazenar os dados localmente, foi utilizada a API LocalStorage do navegador, uma vez que ainda não domino banco de dados. Essa API permite armazenar dados localmente no dispositivo do usuário, tornando a solução bastante prática e fácil de implementar.
